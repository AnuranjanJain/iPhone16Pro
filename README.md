  ## 📂 **Getting Started**

  1. Clone the repository:
     ```bash
     git clone https://github.com/NiranjanKumar001/iPhone-16-Pro.git
     ```
  2. Navigate to the project directory:
     ```bash
     cd iPhone-16-Pro
     ```
  3. Install dependencies:
     ```bash
     npm install
     ```
  4. Run the development server:
     ```bash
     npm run dev
     ```
